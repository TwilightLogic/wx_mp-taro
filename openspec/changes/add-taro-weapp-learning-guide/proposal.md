## Why

Although this repository is already a Taro React starter, it does not yet provide a clear path for an experienced React developer to learn how Taro maps web development concepts to WeChat Mini Program development. Adding a structured learning guide now will shorten onboarding time and turn the project into a practical training sandbox instead of only a scaffold.

## What Changes

- Add a guided learning path for React developers who are new to Taro and WeChat Mini Programs.
- Define a staged curriculum that explains Taro fundamentals, WeChat platform constraints, and the React-to-Taro mental model shift.
- Add hands-on exercises that use the existing project so learners can practice page creation, routing, lifecycle, styling, and API usage.
- Define success criteria for completing the onboarding path so the change can later be implemented consistently.

## Capabilities

### New Capabilities
- `taro-learning-guide`: Provides a structured, practice-oriented learning path for React developers to understand and build WeChat Mini Programs with Taro in this repository.

### Modified Capabilities
- None.

## Impact

- Affects project documentation and onboarding materials for this Taro codebase.
- Uses the existing Taro React starter structure in `src/`, `config/`, and the WeChat Mini Program build workflow from `package.json`.
- Does not require immediate API or runtime changes for the proposal phase, but future implementation may add learning docs, sample pages, and practice tasks.
